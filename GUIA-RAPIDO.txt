1️⃣ ABRIR O SITE
───────────────
Opção A (Mais Rápido):
  - Duplo clique em: index.html

Opção B (Recomendado - com servidor):
  - Abra PowerShell na pasta do projeto
  - Digite: python -m http.server 8000
  - Acesse: http://localhost:8000

2️⃣ EXPLORAR O SITE
──────────────────
✨ Use a navbar no topo para navegar
📍 5 Seções principais:
   • Início - Introdução visual
   • Informações - Conceitos matemáticos
   • Pirâmide 3D - Interativa e calculadora
   • Galeria - Diferentes tipos
   • Grupo - Membros do projeto

3️⃣ INTERAGIR COM A PIRÂMIDE 3D
──────────────────────────────
🖱️ Clique + Arraste = Rotaciona
🔍 Scroll/Roda = Zoom
▶️ Sliders = Ajusta base e altura
🎨 Color Picker = Muda cor
🔲 Wireframe = Mostra estrutura
↺ Reset = Volta à posição inicial

4️⃣ CUSTOMIZAR (Editar o Site)
──────────────────────────────
Usar editor de texto (VS Code, Notepad++, etc):

🔹 Nomes do Grupo:
   → Edite index.html
   → Procure: "SEÇÃO DO GRUPO"
   → Mude nomes, funções, descrições

🔹 Cores (Azul para outra):
   → Edite assets/css/styles.css
   → Procure: :root { --color-...
   → Mude os códigos HEX

🔹 Logo da Turma:
   → Coloque sua imagem em: assets/images/
   → Nome: turma-302-logo.svg (ou .png)
   → Atualize a referência em index.html

🔹 Textos e Conteúdo:
   → Edite direto no index.html
   → Procure pelo texto que quer mudar
   → Modifique e salve

8️⃣ DÚVIDAS FREQUENTES
─────────────────────
P: A pirâmide 3D não aparece?
R: Use um servidor local (python/php/http-server)
   NÃO abra simplemente com clique duplo

P: Os efeitos de scroll não funcionam?
R: Se a página está muito lenta, desative
   "Animações ao Scroll" nas config do navegador

P: Quer adicionar mais informações?
R: Edite o index.html e adicione novos cards
   Copie a estrutura dos existentes

P: Quer mudar a cor do site?
R: Edite :root em assets/css/styles.css
   Mude --color-primary, --color-accent, etc
